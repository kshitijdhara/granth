# Security

If you discover a security vulnerability, please create a GitHub issue in this repository so we can track and triage the report.

Include the following in your report:
- Steps to reproduce
- Affected versions (commit SHA or tag when possible)
- Impact and suggested mitigation
- Proof-of-concept or test case (if available)

We will respond promptly and work with you to remediate the issue. Thank you for helping keep Granth secure.
